Installation

We need a couple of applications to run the Quiety to our local:

Node.js:

It is used to run the build processes. Download from the NodeJs from the official site and install it on your device according to your Operating System (nodejs.org)

Check NodeJs Version:

  • Open the Terminal
  • Run: node -v or node --version
  • If you get the version then the installation of NodeJs is done
$ node --version
$ v14.18.1

Next Packages Install:

Go to project directory then run this command in command line, it will install all packages that we used this template. Make sure that there is no error.


$ npm install

After that your installation process is done.

How to Run?

After installed all packages, you just need to run this script, it will run on your local server on port 'localhost:3000'.


$ npm run dev

How to Build?

As a next.js project Vercel is recommended for deployment and If you want to deploy this project/template on Vercel then your package.json file will look like,

  "scripts": {
    "dev": "next dev",
    "build": "next build",
    "start": "next start",
    "lint": "next lint"
  },

Now you just need to push this project to your GitHub then connect GitHub with vercel account. Then just create new project on vercel by importing from your GitHub, then deploy.


 And if you want to generate static data of this project to deploy cPanel or other hosting service then your package.json file will look like,

  "scripts": {
    "dev": "next dev",
    "build": "next build && next export",
    "start": "next start",
    "lint": "next lint"
  },

now just run npm run build, it will generate all static pages in out folder, then just import that folder to your cPanel to deploy on production.



Folder structure


  • node_modules - Node dependencies libraries
  • package.json - All used npm packages.
  • public - All used static images.
  • src/components - contains all the template-related UI components.
  • src/layout - contains project layout section for all pages and components, also header and footer is included in this folder.
  • src/pages- Here you will find all pages that we use in this project. Root component _app.js and index.js as root Home component.
  • src/assets- contain bootstrap and custom CSS, JS file and font-awesome icons.
  • src/utils- contain all static used data that we used throughout this project by map.

Template Customization

Change Title and Logo

For changing title and logo you just need to go to Layout/layout.js file, then change this section

<title>
          {title
            ? `Quiety | ${title}`
            : 'Quiety - Software & IT Solutions React Template'}
        </title>
        {desc && <meta name="description" content={desc} />}
        <link rel="icon" href="/favicon.png" />



Change meta data

For changing template metadata, just go to src/pages/_document.js file, then change as your need.
